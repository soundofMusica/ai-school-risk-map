
# AI 학교 위험지도 (Streamlit) — 정전/재난 대응

이 프로젝트는 **학교 내부 지도 이미지(PNG/PDF)** + **구역/체크리스트 엑셀** + (선택) 설문/시뮬 입력으로  
구역별 위험을 예측하고 지도에 표시하는 **AI 위험지도 웹앱**입니다.

## 주요 기능
1) 지도 업로드/불러오기(층 선택)
2) 구역 찍기(사각형)로 Zone 폴리곤 생성
3) 위험 점수(0~100) 라벨 입력 → 모델 학습(RandomForest) → 위험지도 오버레이
4) 구역 선택 시 근거(설문/시뮬 + 모델 기여 요인) 표시
5) 인쇄용 체크리스트 PDF 생성(담임/행정실 배포용)

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 데이터 준비
- `AI_학교위험지도_구역_체크리스트.xlsx` : Zones_60 / Checklist_15 시트
- `학교_지도.png` : 지도 이미지(기본 제공)
- `polygons.json` : 구역 폴리곤(앱에서 찍고 저장/다운로드 가능)

## 라벨(risk_label) 채우기 팁
- 설문 평균으로 구역별 위험도를 0~100으로 변환해 입력
- 최소 10개 구역에 risk_label이 들어가면 모델 학습이 시작됩니다.
